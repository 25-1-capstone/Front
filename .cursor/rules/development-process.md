# Expo 개발 프로세스 가이드

## 개발 환경 설정

- Node.js LTS 버전 사용
- Expo CLI 최신 버전 유지
- VS Code + Cursor 확장 사용
- ESLint + Prettier 설정 필수

## 컴포넌트 개발 가이드

1. 컴포넌트 계획

   - 목적과 기능 정의
   - Props 인터페이스 설계
   - 상태 관리 전략 수립

2. 구현 순서

   - 기본 구조 작성
   - 스타일링
   - 기능 구현
   - 최적화
   - 테스트

3. 성능 최적화
   - React.memo 적절히 사용
   - useCallback/useMemo 활용
   - 이미지 최적화
   - 불필요한 리렌더링 방지

## 상태 관리

- 로컬 상태: useState
- 복잡한 상태: useReducer
- 전역 상태: Context API 또는 Redux

## 에러 처리

- try-catch 사용
- 에러 바운더리 구현
- 사용자 친화적 에러 메시지
- 개발용 로깅 설정

## 테스트

- 컴포넌트 단위 테스트
- 통합 테스트
- E2E 테스트 (필요시)
- 성능 테스트
